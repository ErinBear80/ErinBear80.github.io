The goal of this website is to help people learn French.

How to use this website:

It is important to practice French everyday!

It may be helpful to learn with a partner. You can quiz each other and practice speaking together.

Learning a new language takes a strong memory, so work hard.

Make connections to English. How are the languages similar?

Try the practice exercises and visit other websites to practice, too.
